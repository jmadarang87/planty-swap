# Plant Application

An App to swap plants and cuttings locally.

## Technologies Used
- Expo
- React Native, FirebaseAuth

## Features
- Authentication (Sign In, Create Account, Confirm Email, Forgot Password, New
  Password)
- Reusable Components (Button, Input)
- Form Validation
